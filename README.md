| 기술            | 설명                               |
| --------------- | ---------------------------------- |
| Nuxt 3          | Vue.js 프레임워크                  |
| Vue 3           | 프로그레시브 JavaScript 프레임워크 |

## 🚀 시작하기

### 설치

```bash
pnpm install
```

### 개발서버 실행

```bash
# 로컬 환경
pnpm start:local

# 개발 환경
pnpm start:dev

### 빌드

```bash
# 개발 환경 빌드
pnpm build:dev

# 프로덕션 환경 빌드
pnpm build:prod
```
